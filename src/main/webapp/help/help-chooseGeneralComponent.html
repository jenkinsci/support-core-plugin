You can select one of the above values to mark a default set of components. For example, you can select the Default with configuration files value if you want to mark (and collect) the default components of the support bundle and the configuration file components.

You can also select the Custom component and type a hexadecimal value in the auxiliary text box if you want to check some specific components (the components will be checked depending on this hexadecimal value). To get an example of the hexadecimal value, you click in the Select components> button, mark the components that you want and copy the value that you can find in the bottom and right corne of the widget.

Finally, you can click on Select components directly and check the components that you want to collect during the support bundle generation.
